# DentVision1

DentVision1 — проект фронтенда для DentVision (интерфейс/демо). Этот репозиторий содержит исходники и конфигурацию для сборки приложения.

## Быстрый старт

Требования:
- Node.js 18+ (рекомендуется LTS)
- npm (или yarn)

Установка зависимостей:

```bash
npm ci
```

Для разработки (если в проекте есть dev-скрипт):

```bash
npm run dev
# или
npm start
```

Сборка production:

```bash
npm run build
```
```

## Скрипты
Проверьте `package.json` на предмет доступных скриптов. Ожидаемые (если отсутствуют — добавьте):
- `dev` — запуск в режиме разработки
- `build` — сборка проекта
- `lint` — запуск линтера
- `test` — запуск тестов

## CI
Добавлен простой workflow GitHub Actions для запуска установки/линтинга/сборки/тестов при PR и push в main.
