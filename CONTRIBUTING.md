# Contributing to DentVision1

Спасибо за ваш интерес к проекту! Ниже — базовые инструкции для внесения изменений.

1. Форкните репозиторий и создайте новую ветку от `main`:

```bash
git checkout -b feat/your-feature
```

2. Установите зависимости и запустите проект локально:

```bash
npm ci
npm run dev
```

3. Добавьте тесты/линтинг для ваших изменений.

4. Создайте Pull Request в ветку `main` с описанием изменений и шагов для тестирования.

5. Убедитесь, что CI проходит и что ваш PR имеет понятное описание.

Дополнительно:
- Следуйте правилам форматирования кода (ESLint/Prettier если настроены).
- Пишите понятные сообщения коммитов.
